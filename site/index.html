How it works

An agent figured it out once. Nobody should pay for that twice.

Computer-use agents re-discover the same site on every run — full browse, full cost, full minutes. API H splits the work in two:

compile · rare

Explore

One H Computer-Use session opens the site, achieves the goal, and — if you didn't write schemas — discovers them from what it found.

contract · stored

Graduate

The procedure is stored in SQLite as a versioned contract: schemas, HTTP steps or agent prompt, health checks. A procedure for new inputs — not a cached answer.

run · hot

Route

Every request tries the HTTP mapper first (zero agent sessions). No HTTP path? Re-trigger H with a fixed prompt and schema-validate before returning.

// the stored artifact — data/contracts/hn-top-stories-v2.json (excerpt)
{
  "version": 2, "status": "active", "method": "hybrid",
  "http":  { "mapper": "hn_firebase_v0",   // 0 H sessions, ~$0
             "steps": ["topstories.json", "item/{id}.json"] },
  "agent": { "agent_id": "h/web-surfer-pro", // fallback, 1 session
             "prompt_template": "Open {{site}}. Return the top {{limit}}…" },
  "health": { "min_array_length": {"path": "stories", "min": 1} }
}

Every response carries meta.path, latency, and cost — the router never swaps paths silently, and output that fails the schema returns a 502, never bad data.

Engineering honesty

What the eval caught, and what this isn't.

Three real bugs, found by measuring

  • Nullable schemas killed the agent path. ["string","null"] union types crashed the dynamic model builder. Fixed, with a regression test.
  • A greedy slug rule mis-routed a site. Anything named hn* got the Firebase mapper — including Algolia. Specialization is hostname-only now.
  • SQLite pooling deadlocked at 20-way concurrency. Each request held its connection across the upstream await. NullPool fixed it; a burst test guards it.

Honest limits

  • The wrapper trap is real. Craigslist-class sites stay on the agent path — ~103 s and ~9¢ per run — until an HTTP route is reverse-engineered.
  • Graduation is still manual. The $0 paths exist because a mapper was registered. Automatic HAR-based graduation is the unbuilt hard part.
  • n=1 per JS task. One live pass proves the path works, not that it's reliable at scale.
Cache stores answers. A contract stores how to answer — for inputs it has never seen. the judge-trap one-liner
